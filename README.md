# ReVoxi - приложение для передачи звука из микрофона в наушники в реальном времени.
ReVoxi позволяет слышать собственную речь во время разговора, чтения или тренировки произношения. Это помогает лучше контролировать голос, дикцию и темп речи. 
В отличие от диктофона или записи голосовых сообщений, звук передается в реальном времени с минимальной возможной задержкой.

## Возможные сценарии применения:
- Тренировка темпа и громкости речи;
- Репитиции публичных выступлений;
- Развитие навыков произношения;
- Тренировка иностранного произношения;
- Самостоятельная работа с голосом.

## Архитектурные решения: 
- Гибридная разработка для достижения минимальной задержки: использование нативного Kotlin-кода для аудиопотока через Platform Channels и реализация кроссплатформенного интерфейса с помощью dart/flutter;
- Централизованная обработка ошибок с нативного и сценарного слоев;
- Русская и английская локализация;
- Интеграция с телеграм-ботом для отправки отчетов об ошибках;
- Централизованная система логирования;
- Использование MVVM и принципов чистой архитектуры.
